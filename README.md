# netMusic

## 项目安装
```
yarn install
```

### 编译项目及挂载运行
```
yarn serve
```

### 项目打包
```
yarn build
```

此项目为仿网易云音乐的移动端音乐app，通过https://neteasecloudmusicapi.vercel.app/#/提供的接口可实现歌单，专辑、歌词和搜索等功能
